Dino Run

Personal Project

I developed a Dino Run game using Pygame, incorporating dynamic graphics, sound effects, and user interactions. The game initializes Pygame, sets up the display, and loads assets like images and sounds. It manages game states (ready, active, game over) through a global variable and handles events such as keyboard and mouse inputs. The main character, defined in the Dino class, has states for running, jumping, and ducking. Enemy obstacles are managed by the Cactus and Bird classes, while the Clouds class adds background elements. Collision detection and scoring mechanisms ensure a challenging gameplay experience.
